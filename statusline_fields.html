Claude Code 狀態列 — 全部可用參數

stdin JSON 裡的所有欄位。想顯示哪個就挑哪個,然後打 /statusline …

★ = 可能不存在或為 null(腳本要做防呆,jq 用 // 0 或 // empty)
model — 模型
model.display_name模型顯示名(Opus)
model.id模型 ID(claude-opus-4-8)
workspace — 目錄 / 工作區
cwd目前工作目錄(同 current_dir)
workspace.current_dir目前目錄(建議用這個)
workspace.project_dir啟動時的專案根目錄
workspace.added_dirs/add-dir 加進來的額外目錄(陣列)
workspace.git_worktree ★在 git worktree 裡時的 worktree 名
workspace.repo.host ★repo 主機(github.com)
workspace.repo.owner ★repo 擁有者(anthropics)
workspace.repo.name ★repo 名稱(claude-code)
cost — 花費 / 時間
cost.total_cost_usd本 session 估算花費(USD)
cost.total_duration_mssession 總時長(ms)
cost.total_api_duration_ms等 API 回應的總時間(ms)
cost.total_lines_added新增行數
cost.total_lines_removed刪除行數
context_window — Context 用量
context_window.used_percentage ★已用 %(最常用)
context_window.remaining_percentage ★剩餘 %
context_window.context_window_size上限(200000 或 1000000)
context_window.total_input_tokens目前 context 的 input token 數
context_window.total_output_tokens目前 output token 數
…current_usage.input_tokens ★input token
…current_usage.output_tokens ★output token
…cache_creation_input_tokens ★寫入 cache 的 token
…cache_read_input_tokens ★讀 cache 的 token
exceeds_200k_tokens是否超過 20 萬 token(true/false)
推理 / 思考
effort.level ★推理 effort(low/medium/high/xhigh/max)
thinking.enabled是否開啟 extended thinking
rate_limits — 額度 (限 Pro/Max)整段 ★
rate_limits.five_hour.used_percentage5 小時額度已用 %(0–100)
rate_limits.five_hour.resets_at5 小時窗口重置(Unix 秒)
rate_limits.seven_day.used_percentage7 天額度已用 %
rate_limits.seven_day.resets_at7 天窗口重置(Unix 秒)
Session 資訊
session_idsession 唯一 ID(適合當 cache key)
session_name ★自訂名(--name / /rename 設的)
transcript_path對話記錄檔路徑
versionClaude Code 版本
output_style.name目前輸出風格名稱
其他(特定情境才出現)整段 ★
vim.modevim 模式(NORMAL/INSERT/VISUAL…)
agent.name--agent 跑時的 agent 名
pr.number目前 branch 的 PR 編號
pr.urlPR 連結
pr.review_statePR 審查狀態(approved/pending…)
worktree.name--worktree session 的 worktree 名
worktree.pathworktree 絕對路徑
worktree.branchworktree 的 branch
worktree.original_cwd進 worktree 前的目錄
worktree.original_branch進 worktree 前的 branch
